KEY FEATURES

- Full system scan with detailed results
- 75+ real system optimizations (platform-specific)
- Category-based fixes:
  - Performance

  - Disk & Storage
  - Startup
  - Power

- Gaming / FPS optimization mode (session-based, auto-revert)

- Cross-platform support (Windows, Linux, macOS)
- No background services
- No telemetry
- No fake performance claims

--------------------------------------------------

GAMING MODE (IMPORTANT NOTES)

Gaming optimizations are:
- Session-only
- Automatically reverted
- CPU, scheduler, and memory focused

GPU-limited games may not see FPS gains.
SystemPulse Optimizer does not modify drivers or apply unsafe kernel tweaks.

--------------------------------------------------

SAFETY & TRANSPARENCY

- Every optimization shows:
  - What will change
  - Risk level
  - Whether a restart is required
- Backups are created before risky changes
- Undo is available for all supported operations
- Advanced optimizations require explicit user confirmation

SystemPulse Optimizer never hides actions or applies changes silently.
