# SystemPulse Optimizer - Binaries Only

Prebuilt desktop binaries are provided here. Source code is not published in this repository.

## Download
- macOS: systempulse-mac.tar.gz (contains systempulse-mac and SHA256SUMS.txt)

## Verify
- macOS: `shasum -c SHA256SUMS.txt`

## Run
- macOS: `tar -xzf systempulse-mac.tar.gz && chmod +x systempulse-mac && ./systempulse-mac`
  - If macOS blocks launch, right-click the app -> Open once, or allow under System Settings -> Privacy & Security.

## Notes
- Session-only tweaks; changes auto-revert when the session ends.
- Results vary by hardware and workload; no guaranteed FPS gains.
